# ORGANIZER
**Simple task manager (server-side)**

### Technologies
- Java
- Spring: Security | JPA 
- PostgreSQL
- Docker
- Swagger

### Functionality
- Register and auth
- Create, edit and delete tasks
- Comment tasks
- Receive notifications about your tasks (!!!sockets)
- Administration

### Start guide
1. Clone the repo
2. Build with maven
3. Run docker compose
4. ...
5. Have fun!
